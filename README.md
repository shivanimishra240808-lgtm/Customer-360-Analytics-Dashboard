# Customer 360 Analytics Dashboard

## 📊 About the Project
I built this project to understand how customers behave, how they spend, and which segments generate the most revenue. The goal was to simulate a real business scenario where data is used to make decisions.

## 🛠 Tech Stack
- SQL for data analysis
- Python (Pandas) for data cleaning
- Power BI for dashboard creation

## 📁 Project Structure
- data → contains raw datasets (customers, orders, products, payments)
- sql → SQL queries used for analysis
- python → data cleaning and preprocessing script
- dashboard → Power BI dashboard file (.pbix)
- screenshots → dashboard preview images

## 📌 Key Learnings
- Identified high-value vs low-value customers
- Observed that a small segment contributes to most revenue
- Analyzed country-wise performance (UK leading)
- Found consistent growth pattern in monthly sales
